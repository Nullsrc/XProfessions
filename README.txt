The is a mod for Cataclysm: Dark Days Ahead. It adds several new professions.
===============================================
How to install: replace the professions.json file in your json folder. In windows, the folder path should look like: Cataclysm\data\json\professions.json
===============================================
*Entrepreneur
*Rifle Hunter
*Miner
*Martial Artist
*Delinquent
*Spelunker
*Factory Worker
*Cowboy
*Welder
*Camper
*Biggest Fan
===============================================
Version 1.0: Original Release, included: Entrepreneur, Rifle Hunter, Miner, Martial Artist, Delinquent, Spelunker, Factory Worker, Cowboy, Welder, Camper, and Biggest Fan
